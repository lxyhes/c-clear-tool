# C-Clear-Tool (C盘智能清理工具)

这是一个基于 Python 的轻量级 Windows C 盘清理工具，旨在安全地识别和清理系统垃圾、应用缓存，并协助发现大文件。

它采用 **"安全白名单" + "特征识别"** 策略，能够在清理垃圾的同时，最大程度保护您的重要数据（如游戏存档、浏览器书签、聊天记录等）。

## ✨ 核心功能

*   🧹 **智能扫描**: 自动扫描 `AppData`（Local/Roaming）下的缓存、临时文件、日志和崩溃报告。
*   🏢 **软件识别**: 自动识别并标注常见软件名称（如 `[微信] Tencent WeChat`, `[谷歌] Chrome`, `[Adobe] Common`），拒绝乱码目录。
*   🛡️ **安全机制**: 
    *   **白名单保护**: 自动避开 `User Data`, `SaveGames`, `Profile` 等敏感目录。
    *   **占用跳过**: 删除时自动检测文件占用，防止程序崩溃。
*   🐘 **大文件查找**: 独立模块扫描下载、桌面、文档、视频目录下的超大文件 (>100MB)，帮您快速定位空间杀手。
*   ⚡ **极速体验**: 采用多线程并行扫描 + `os.scandir` 优化遍历算法，速度飞快。
*   📂 **便捷交互**: 
    *   **右键菜单**: 在扫描结果上**右键点击**，可直接打开所在文件夹查看。
    *   **可视化列表**: 清晰展示每个项目的路径和占用大小。

## 🚀 如何使用

### 前置要求
*   Windows 10 / 11
*   Python 3.x (已添加到环境变量)

### 启动方式
1.  下载本项目代码。
2.  找到 **`start_cleaner.bat`** 文件。
3.  **右键点击**该文件，选择 **"以管理员身份运行"**。
    *   *注意：虽然普通运行也能工作，但管理员权限能清理更多系统级临时文件（如 C:\Windows\Temp）。*

### 操作指南
1.  **垃圾清理**:
    *   点击 **"开始扫描"**。
    *   勾选您想要删除的项目（默认会勾选安全的缓存项）。
    *   如果拿不准，**右键点击**项目 -> **"打开所在文件夹"** 确认内容。
    *   点击 **"立即清理选中项"**。

2.  **大文件查找**:
    *   切换到 **"🐘 大文件查找"** 选项卡。
    *   点击 **"开始查找大文件"**。
    *   找到不需要的大文件后，**右键点击** -> **"打开所在文件夹"**，然后手动删除。
