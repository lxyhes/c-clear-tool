# 🚀 C-Clear-Tool (C盘深度清理助手)

### **“深度清理，不留余地；优雅离职，不留痕迹。”**

[![Python](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/) [![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)]() [![License](https://img.shields.io/badge/license-MIT-green.svg)]()

**C-Clear-Tool** 是一款专为职场专业人士设计的开源 Windows C 盘深度清理工具。它不仅能解决日常 C 盘爆满的问题，更深度聚焦于**社交办公软件的精细化清理**以及**离职时的物理级隐私粉碎**。

---

## 🔥 为什么要选择 C-Clear-Tool？

市面上的清理工具很多，但 C-Clear-Tool 在以下场景具有绝对优势：
*   ✅ **精细化社交专清**：按账号分类，精准识别图片、视频、文件，支持选择性删除。
*   ✅ **离职隐私粉碎**：不仅是删除，更是**物理覆盖粉碎**，防止数据恢复。
*   ✅ **全方位足迹抹除**：涵盖浏览器密码、SSH私钥、数据库配置、邮件存档、指令历史及网络印记。
*   ✅ **极速雷达引擎**：24 线程并发扫描，智能避让系统冗余目录，性能压榨至极限。

---

## ✨ 核心特性

### 1. 🛡️ 离职专清完全体 (Fully Resignation Shredder)
*专为开发者、运维及行政人员设计的终极脱敏方案：*
*   🔑 **凭据粉碎**：粉碎浏览器保存的所有密码、Cookies，清空 Windows 凭据管理器。
*   💻 **技术脱敏**：抹除 SSH 私钥、Git 全局配置、**PowerShell 指令历史**（含明文密码风险）。
*   🗄️ **数据库与终端**：清理 Navicat、XShell、FinalShell 的连接记录与查询历史。
*   📧 **邮件存档**：锁定并粉碎 Outlook、Foxmail 存储在本地的 PB 级邮件存根。
*   🌐 **网络印记**：自动刷新 DNS、ARP 缓存，断开并清除所有网络共享连接记录。
*   📋 **审计报告**：清理后自动在桌面生成《离职安全审计报告》，交接更有底气。

### 2. 💎 精细化社交办公专清 (Multi-Account Radar)
*深度适配国内主流协作软件：*
*   **覆盖软件**：微信、**企业微信**、QQ、钉钉、飞书。
*   **维度拆解**：针对每个账号独立列出【图片、视频、文件、缓存】，解决“想删视频但想留文档”的痛点。

### 3. 🚀 极速雷达 2.0 引擎
*   ⚡ **原子级分治扫描**：任务拆解至文件夹级，实现极致的 I/O 并发。
*   🧠 **智能避让逻辑**：自动跳过 `WinSxS`、`System32` 等无效区域，扫描提速 80%。
*   🔓 **进程自动解锁**：清理前自动检测并一键关闭占用文件的相关程序。

### 4. 🎨 现代 Fluent UI
*   采用系统原生 **彩色 3D 符号图标**，高清不失真，完美适配 Windows 11。
*   分片渲染技术，确保在处理海量文件清单时界面依然丝般顺滑。

## 🛠️ 快速开始

1.  **下载/克隆** 本仓库。
2.  双击运行 **`start_cleaner.bat`**。
3.  **获取权限**：程序会自动请求管理员权限。
4.  **选择模式**：
    *   🧹 *智能清理*：日常维护。
    *   💬 *社交专清*：精细化清理聊天文件。
    *   🔥 *离职专清*：物理粉碎身份印记与网络记录。

---

## ⚙️ 高级配置

### 自定义清理路径
如果默认扫描无法覆盖您的特殊需求，您可以在程序根目录下编辑或创建 **`custom_paths.txt`** 文件。
*   **格式**：每行输入一个绝对路径。
*   **作用**：程序启动时会自动加载这些路径并纳入扫描范围。
