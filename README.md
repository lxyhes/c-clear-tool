# 🚀 C-Clear-Tool (C盘深度清理助手)

### **“深度清理，不留余地；优雅离职，不留痕迹。”**

[![Python](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/) [![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)]() [![License](https://img.shields.io/badge/license-MIT-green.svg)]()

**C-Clear-Tool** 是一款专为职场专业人士设计的开源 Windows C 盘深度清理工具。不同于传统的通用清理软件，它深度聚焦于**社交办公软件的死角清理**以及**离职时的极端隐私粉碎**。

---

## 🔥 为什么要选择 C-Clear-Tool？

市面上的清理工具很多，但它们往往：
*   ❌ 扫不到微信、QQ 动辄几十 GB 的加密数据库。
*   ❌ 无法识别钉钉、飞书等企业级软件的冗余文件。
*   ❌ 扫描慢，面对海量小文件时界面卡死。
*   ❌ **离职时，无法一键彻底擦除浏览器密码、SSH 密钥和邮件存根。**

**C-Clear-Tool 解决了以上所有问题。**

---

## ✨ 核心特性

### 1. 🛡️ 离职专清模式 (Resignation Shredder)
*专为开发者与行政办公人员设计，一键粉碎电脑上的身份印记：*
*   🔑 **凭据清理**：自动锁定并粉碎浏览器保存的所有密码、Cookies 和访问历史。
*   💻 **技术脱敏**：清理 SSH 远程服务器私钥 (`id_rsa`)、Git 全局配置、PowerShell 指令历史。
*   📧 **邮件存档**：扫描并清理 Outlook、Foxmail 存储在本地的 PB 级邮件存根。
*   🧩 **IDE 历史**：擦除 VS Code、JetBrains 记录的项目足迹和本地修改历史。

### 2. 💬 社交办公专清 (Social App Radar)
*深度适配国内主流通讯软件，不仅是清理缓存：*
*   **全盘雷达**：利用“特征签名”技术全盘检索 **微信、QQ、钉钉、飞书** 的存储位置（无论您迁移到哪个盘）。
*   **细粒度展示**：按账号列出图片、视频、文件、数据库占用，支持精准剔除。

### 3. 🚀 极速雷达 2.0 引擎
*性能怪兽，极致并发：*
*   ⚡ **原子级分治扫描**：将扫描任务拆解至文件夹级，开启 **24 线程** 并发检索，瞬间填满 SSD I/O。
*   🧠 **智能避让逻辑**：自动跳过 `WinSxS`、`System32` 等无效的系统死角，节省 80% 的扫描时间。

### 4. 🎨 现代 Fluent UI
*   高清彩色 3D 符号装饰，适配 Windows 11 视觉风格。
*   分片渲染技术，即使扫描出 10 万个文件，界面依然丝般顺滑。

---

## 🛠️ 快速开始

1.  **下载/克隆** 本仓库。
2.  双击运行 **`start_cleaner.bat`**。
3.  **获取权限**：程序会自动请求管理员权限以清理系统级垃圾。
4.  **选择模式**：
    *   🧹 *智能清理*日常维护。
    *   💬 *社交专清*：深度清理通讯软件。
    *   🔥 *离职专清*：终极身份脱敏。
